Zorgplanner

👋 Welkom bij Zorgplanner

Vul jouw naam in zodat je jezelf snel kunt kiezen als chauffeur of meerijder.

Nieuwe afspraak

Gebruik 09:00 of 0900. Geldig: 00:00–23:59.

    ⚙️ Beheer

    👤 Namen

    Namen voor chauffeur en meerijder. Vrij te typen of hier toe te voegen.

    🧸 Oppas / opvang opties

    Veelgebruikte opties. Je kunt ook altijd vrij typen bij de afspraak.

    🔄 Delen met familie

    Deelcode: kopieer en plak om gegevens tussen apparaten over te zetten. Let op: dit is nog geen live familiesynchronisatie.

    💾 Back-up als bestand

    Exporteer als JSON-bestand of importeer een eerder gemaakte back-up.

    Importeren overschrijft de huidige gegevens op dit apparaat. Maak dus eerst een back-up als je twijfelt.

    🗑️ Afspraak verwijderen?

    Dit kan niet ongedaan worden gemaakt.

    👨‍👩‍👧‍👦 Familie verbinden

    Voer de familiecode in om afspraken te delen en samen te werken. Iedereen met dezelfde code ziet dezelfde afspraken.

    De code is hoofdletterongevoelig. Vraag de code op bij de beheerder van de Zorgplanner.
